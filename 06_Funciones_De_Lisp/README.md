# 20 funciones basicas de Lisp

### 1. Función suma (+)
Permite sumar 2 o mas numeros que se manden como parametro.

**Sintaxis**
```lisp
(+ a b c d e f)
```

**Ejemplo**
```lisp
(+ 2 2 4)

8
```

### 2. Función resta (-)
Al primer numero que se mande como parametro le resta todos los demas que encuentren en la función.
**Sintaxis**
```lisp
(- num b c d e f)
```

**Ejemplo**
```lisp
(- 2 3 4)

-5
```
